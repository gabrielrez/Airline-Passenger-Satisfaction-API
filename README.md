# Airline Passenger Satisfaction API

API em FastAPI para prever a satisfação de passageiros com base em dados de um modelo treinado em Machine Learning.

---

## Como rodar o projeto

<strong>1. Clone o repositório</strong>
```
  - git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.
  - cd SEU_REPOSITORIO
```

<strong>2. Crie um ambiente virtual</strong>
```
python -m venv venv
source venv/Scripts/activate
```

3. Instale as dependências
```
pip install -r requirements.txt
```

4. Rode o servidor
```
python -m uvicorn main:app --reload
```

5. Acesse http://127.0.0.1:8000/docs
